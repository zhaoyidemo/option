# 使用示例

本文档通过实际案例演示双币理财看板的使用流程。

## 场景一：BTC 低买（看跌期权）

### 情况描述

你有 10,000 USDT，想通过双币理财获取收益。

### 操作步骤

#### 1. 配置初始资产

点击"资产配置"：
- USDT: `10000`
- BTC: `0`
- ETH: `0`

#### 2. 创建第一笔交易

在币安创建 BTC 低买交易：

```
平台: 币安
币种: BTC
方向: 低买（看跌期权）
投入数量: 1000
投入币种: USDT
行权价: 60000
到期时间: 2024-01-20 08:00
年化收益率: 12
未行权收益: 5.5 USDT
行权得到: 0.01667 BTC
```

### 结果分析

#### 场景 A：到期价格 $62,000（未行权）

- 结算价格 > 行权价（62000 > 60000）
- **不行权**
- 得到：1000 + 5.5 = **1,005.5 USDT**
- 收益：+5.5 USDT

#### 场景 B：到期价格 $58,000（行权）

- 结算价格 < 行权价（58000 < 60000）
- **行权**
- 得到：**0.01667 BTC**（价值约 967 USDT）
- 账面亏损：-33 USDT
- 但现在持有 BTC，可以继续做高卖

## 场景二：复投 - 行权后做高卖

### 接上一个场景 B

你现在持有 0.01667 BTC，想继续赚取收益。

### 创建第二笔交易（复投）

在欧易用 BTC 做高卖：

```
平台: 欧易
币种: BTC
方向: 高卖（看涨期权）
投入数量: 0.01667
投入币种: BTC
行权价: 65000
到期时间: 2024-01-25 08:00
年化收益率: 15
未行权收益: 0.00005 BTC
行权得到: 1083.55 USDT
```

### 结果分析

#### 场景 A：到期价格 $63,000（未行权）

- 结算价格 < 行权价（63000 < 65000）
- **不行权**
- 得到：0.01667 + 0.00005 = **0.01672 BTC**
- 收益：+0.00005 BTC（约 +3.15 USDT）

#### 场景 B：到期价格 $67,000（行权）

- 结算价格 > 行权价（67000 > 65000）
- **行权**
- 得到：**1,083.55 USDT**
- 总收益计算：
  - 初始：10,000 USDT
  - 现在：9,000 + 1,083.55 = 10,083.55 USDT
  - 总收益：+83.55 USDT

## 场景四：自动结算示例

### 交易信息

```
平台: 币安
币种: BTC
方向: 低买
投入: 1000 USDT
行权价: 60000
到期时间: 2024-01-20 08:00
```

### 自动结算流程

1. **2024-01-20 08:00** - 到期时间到达
2. **08:05** - Cron 任务触发（每 5 分钟检查一次）
3. **系统获取 CMC 价格**：BTC = $62,000
4. **判断是否行权**：62000 > 60000，不行权
5. **更新交易状态**：
   - status: `settled`
   - exercised: `false`
   - settlementPrice: `62000`
   - outputAmount: `1005.5`
   - outputCurrency: `USDT`

### 查看结果

在"已结算"列表中看到：

```
✓ 币安 BTC 低买 | 未行权 | +5.5 USDT
结算价: $62,000
```

## 实用技巧

### 1. 最大化收益

- **高频复投**：到期后立即创建新交易
- **分散平台**：对比币安和欧易的收益率
- **灵活调整**：根据市场走势选择低买/高卖

### 2. 风险控制

- **行权价设置**：
  - 低买：选择低于当前价 5-10% 的行权价
  - 高卖：选择高于当前价 5-10% 的行权价
- **分批投入**：不要一次性投入所有资金
- **止损策略**：被行权后可以反向操作

### 3. 数据监控

- **每日检查**：查看实时价格和净值变化
- **分析趋势**：关注哪个平台、哪个币种收益更高
- **及时调整**：根据市场情况调整策略

## 常见问题

### Q1: 为什么我的收益和预期不一样？

**原因**：
- 行权导致币种变化，价格波动影响净值
- 多次复投的复利效应
- 不同时间点的价格差异

### Q2: 如何选择行权价？

**建议**：
- **激进策略**：行权价接近当前价（收益率高，行权概率大）
- **保守策略**：行权价远离当前价（收益率低，行权概率小）

### Q3: 被行权了怎么办？

**应对方法**：
- 如果得到币，可以做高卖
- 如果得到 USDT，可以做低买
- 继续复投，形成资金循环

### Q4: 如何提高整体收益率？

**技巧**：
- 选择年化收益率高的交易
- 缩短持有周期，加快复投频率
- 利用两个平台的差价套利
